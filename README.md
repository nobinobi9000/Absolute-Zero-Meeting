# ねこおじさん～絶対零度会議室～（会議タイマー）

会議の雰囲気を一瞬で凍りつかせる（かもしれない）「おやじギャグ」と、たまに「ためになる話」を届けてくれるWebアプリです。

## 特徴
- **タイマー機能**: 会議の時間を設定してカウントダウン。終了時には猫のおじさんがお知らせします。
- **おやじギャグ機能**: 「本日の終わりの一言」ボタンを押すと、ランダムでおやじギャグ（80%）か、ためになる名言（20%）が表示されます。
- **演出**: タイマー終了時やセリフ表示時に、状況に応じた専用のサウンドが流れます。

## 使い方
1. 公開URLにアクセスします。
2. 「+分」ボタンでタイマーを設定します。
3. 「スタート」ボタンでカウントダウンを開始します。
4. 時間になると、猫のおじさんが現れておやじギャグ（または名言）を披露します。
5. 会議の途中で空気を変えたいときは、「本日の終わりの一言」ボタンを押してください。
